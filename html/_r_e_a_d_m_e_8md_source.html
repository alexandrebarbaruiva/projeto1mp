8 ## Requirements
9 
10 You must have a C++ compiler (4.9 or above) installed on the machine.
11 The operating system can be Linux, Windows or MacOS.
12 Compiler installation:
13 - [Windows.](https://cs.calvin.edu/courses/cs/112/resources/installingEclipse/cygwin/)
14 - [Mac.](https://www.mkyong.com/mac/how-to-install-gcc-compiler-on-mac-os-x/)
15 - [Linux.](https://askubuntu.com/questions/348654/how-to-install-g-compiler)
16 - [Linux 4.9.](https://askubuntu.com/questions/428198/getting-installing-gcc-g-4-9-on-ubuntu)
17 
18 For coverage, gcovr is used. Go to http://gcovr.com/ for installation process.
19 It's used [valgrind](http://valgrind.org/) to check use of memory. [Installation process here.](http://valgrind.org/docs/manual/dist.install.html)

21 ## Installing/Compiling
22 
23 Download the entire project directory, open the terminal or the equivalent of your machine, access the directory in which the project was downloaded using
24 
25 ```
26 cd /YourDirectory/YourFolder/projeto1mp
27 ```
28 Insert your input according to the requisites found here: [PDF Document](mp_t1.pdf).
29 Then compile using make.
30 
31 ```
32 make
33 ```
34 In case your gcc is below 4.9, install using the [link provided](https://askubuntu.com/questions/428198/getting-installing-gcc-g-4-9-on-ubuntu)
35 and use
36 ```
37 make stable
38 ```
39 For tests, run
40 ```
41 make test
42 ```
43 or
44 ```
45 make stabletest
46 ```
47 For checking performance
48 ```
49 make perf
50 ```
51 or
52 ```
53 make stableperf
54 ```
